可视化模板设计器

从左侧工具箱拖拽元素到画布，支持：文本、图片、线条、矩形、表格、条形码、二维码、数据字段等。
快捷键：Ctrl+Z 撤销, Ctrl+Y 重做, Delete 删除, 方向键 微调位置。

🎯 自定义打印范围

可选择打印整个页面、特定元素（如表格、表格区域）或排除某些元素（如广告、导航栏），避免冗余内容出现在打印输出中。

🎨 样式控制与媒体查询

支持通过 CSS @media print 定义专用于打印的样式，隐藏非必要元素、调整字体大小、边距等，确保打印效果整洁美观。

🌐 跨浏览器兼容性

兼容主流浏览器（Chrome、Firefox、Edge、Safari 等），自动检测浏览器环境并应用兼容性修复。

📁 多格式导出支持

支持将打印内容导出为 PDF（使用 jsPDF）、PNG、JPG（使用 html2canvas）、HTML 等格式，便于存档或分享。

🔗 动态数据绑定

插件可将后端数据（如订单信息、用户资料）动态填充到预设模板中，适用于票据、发票、报表等场景。

🖨️ 静默打印与批量打印

支持无需弹出对话框的"静默打印"（需浏览器扩展或本地服务）；同时支持批量处理多个打印任务和打印队列管理。

🎮 可视化模板设计器

提供拖拽式设计器，允许非技术人员通过界面配置打印模板，支持撤销/重做、属性编辑、网格吸附等功能。

📝 表单与 Canvas 元素支持

插件能正确捕获并打印用户输入的表单值（文本框、复选框、单选框、下拉框等）以及 Canvas 绘图内容，确保打印完整性。

📦 助手函数系统

内置多种数据格式化助手函数：formatDate（日期格式化）、formatNumber（数字格式化）、formatCurrency（货币格式化）、uppercase/lowercase（大小写转换）等。

🧩 条件与循环渲染

模板系统支持 {{#if}} {{/if}} 条件渲染和 {{#each}} {{/each}} 循环渲染，灵活处理复杂数据展示需求。

📐 网格与标尺

模板设计器提供网格吸附、像素级定位、顶部/左侧标尺显示，帮助精准对齐元素，制作专业级打印模板。

🔄 撤销/重做

完整的历史记录管理，支持最多50步撤销/重做操作，让模板设计更加安心。
